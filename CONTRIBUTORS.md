# Contributors

Everyone listed here is credited as a co-author on every public release
commit (via `Co-authored-by:` trailers added by `tools/release.py`), which
is what makes GitHub's contributors graph on the public mirror reflect the
people behind the project — the mirror's history is one squashed commit per
release, so trailer credit is the mechanism, not per-commit authorship.
